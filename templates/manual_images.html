Syntax

img-container r_ratioValue effectClass

Example

Rules

  • img elements need to be wrapped in a div with the img-container class. This class get by default 100% of the parent's width. You can counter this by adding a width utility class.
  • To apply a specific ratio to the picture, add a ratio class to the img-container:
    • r_16-9
    • r_2-1
    • r_1-1
    • r_1-2
    • r_4-3
  • By default, pictures in a img-container will have an object-fit css property set to cover. To change this, add a fitting class:
    fit--contain|scaledown|unset|none
  • A picture's natural ratio can differ from its container's, resulting in truncated content. For exemple, if you set a squared picture (ratio 1:1) in an img-container with a 16:9 ratio, the top and bottom part of the picture will be cropped to fit the container's width:
    picture doesn't fit the container, so its top and bottom are cropped to stay centered
    By default, pictures are centered in their container. You can alter the picture positionning by adding a vertical and/or an horizontal position class: img--top|center|bottom img--left|center|right

    with img--top class
  • Pictures can be rounded by adding the rounded-circle class to the container. Be aware that this will force a 1:1 ratio on the picture:
  • grayscale class add a basic grey filter to the picture
  • You can define basic effects on hover, by adding an effect class to the img-container:
    • zoominzoomout
    • fadetocolorfadetogrey
